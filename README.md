# number-guessing-game-withhighscore

A simple command-line number guessing game with a highscore leaderboard stored in a CSV file.

## Features

| Command    | Description           |
|------------|-----------------------|
| start      | Start a new game      |
| highscores | View the leaderboard  |
| clear      | Clear the leaderboard |
| quit       | Exit the game         |

## How to run
```
git clone https://github.com/apathydevil/number-guessing-game-withhighscore.git
cd number-guessing-game-withhighscore
python main.py
```

## What I learned
This was my second project. I learned CSV file I/O, the random module, sorting with lambda functions and building a local leaderboard system. Also implemented a difficulty system with a fallback default case for invalid inputs.
